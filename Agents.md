# Reddit Community Pulse Scraper

The Reddit Community Pulse Scraper tracks popular posts from specific subreddits to identify trending keywords and emerging memes. This scraper is useful for identifying content trends in real-time.

## Features:
- Fetches top "hot" posts from Reddit subreddits.
- Extracts and analyzes keywords from post titles.
- Outputs data on post URLs, titles, upvotes, and trending keywords.

## Inputs:
1. `subreddits`: List of subreddits to monitor.
2. `keywordAnalysis`: Number of top keywords to extract per subreddit.
3. `maxPostsPerSubreddit`: Number of hot posts to fetch per subreddit.

## Outputs:
- Subreddit
- Post Title
- Keywords Extracted
- Post URL
- Upvotes

## Use Cases:
- Trend Analysis: Track emerging discussions or meme trends on Reddit.
- Content Creation: Find inspiration for viral ideas in active communities.
- Marketing Insights: Understand what Reddit users are engaging with the most.
